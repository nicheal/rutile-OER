This directory contains all of the finished calculations required for fully analysis of the paper "A linear response, DFT+U study of trends in the oxygen evolution activity of transition metal rutile dioxides" by Zhongnan Xu, Jan Rossmeisl, and John R Kitchin. To use this repository, download the supporting information file and run the scripts present in either 'supporting-information.pdf' or 'supporting-information.org'.
